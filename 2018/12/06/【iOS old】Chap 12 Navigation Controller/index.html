Segues 需要标识符

当应用的页面越来越多，我们需要给每个 segue 起名字以区分各自。

Scenes and Segues

连接多个视图控制器

Scenes: refer to the on-screen content.

Segues: 连接两个 Scenes, represent the transition from one scene to another. A segue object contains both the source and destination view controllers involved in the transition.

Adding a Detail View Controller

要实现点击一行进入详细页，我们要添加一个新的视图控制器，并与主控制器建立连接，传递信息。

连接: ctrl and drag datacell to Detail View Controller

Segue 的几种显示方式

  • **Show - **默认的，当前页面被推走，带导航栏和返回按钮
  • **Show detail - **取代当前，无导航栏和返回按钮
  • **Present modally - **从底部向上动态覆盖整个页面。在 Add 时比较常用
  • **Present as popover - **iPad 上常用
