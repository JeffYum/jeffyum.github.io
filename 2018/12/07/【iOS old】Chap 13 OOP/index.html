OOP 面向对象编程的一些基本概念

Properties Functionalities
汽车 颜色，型号，最大速度，制造商 加速，刹车，转向
UIButton 背景，大小，颜色，字体 检测 touch

Method: 提供对象的功能，通常一个方法与对象的一个特定功能相关。

函数(function)和方法(method)有什么区别？功能上没啥差别，函数定义在类中被称为方法。

Class: 类是创建对象的蓝图或模型。通常，一个类由 properties 和 methods 组成。例如，我们定义一个 Course 类，它包含一些 properties，如名称，课程编号，总人数。

这个类代表着课程的模型，我们用它创建不同的课程，如 iOS Programming course with course code IPC101, Cooking course with course code CC101 等等。在这里，”iOS Programming course” 和 “Cooking course” 是 Course 类的对象。我们通常把一门课程称为 Course类的一个 instance. 为简单起见，术语 instance 和 object 有时可以交换。

A blueprint for a house design is like a class description. All the houses built from
that blueprint are objects of that class. A given house is an instance.
