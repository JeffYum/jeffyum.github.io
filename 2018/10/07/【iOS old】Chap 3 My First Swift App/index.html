There are not many contents in this chapter. We talk about the basic elements of user interface. Then, we creat a button in the original view, and customize it. Last, we work in ViewController.swift to creat a pop-up after clicking the button and connect them together.

UI 部分

Scene： 故事板中一个屏幕表示一个 view controller 及其 view.

view： UI 基本模块。例如 storyboard 中自带的 view 是放其他 buttons, labels, image views 的容器。

view controller： 管理与之关联的 view 和 subviews(e.g. button and label).

Command+= can let Xcode resize the button, 让文字显示完整。

让 button 做出反应

在 ViewController.swift 的 ViewController 类中添加 showMessage(sender: UIButton) 方法

建立联系

我们需要将故事板中的「Hello World」Button 与刚刚创建的方法联系起来。

Simply 按住 Ctrl 和鼠标左键拖拽到控制器图标
