Auto Layout

发表于 2018-10-11 更新于 2022-08-09 分类于 iOS Disqus：

This is the first article that I use English totally, because the content in this chapter is talking about some apparent working, doesn’t involve any logical code. In short, Auto Layout is to prevent the objects in the view from moving everywhere. We discussed how to set and edit Auto Layout and some theories about it.

An adaptive UI that responds appropriately to changes in screen size and device orientation.

Constraint-based Layout System

There are two approaches to add them:

  1. layout button at the lower right corner
  2. Control-drag, hold the control key, drag the mouse to the orient that you want to set a constraint. When the view become blue, release clicking, a pop-over menu will show.

Align: e.g. vertically center, horizontally center.

Spacing constraint: e.g. The label is 20 points away from the bottom

Safe Area: the entire view excluding the status bar, navigation bar or tab bar. It can make the objects layout correctly even if there are all kinds of bars.

Editing Constraints

Change the properties of constraints including relation, constant and priority.

  1. Attributes inspector
  2. Double click the constraint
