发表于 2019-03-17 更新于 2022-08-09 分类于 Python Disqus：

模块，包，import

模块里面可以定义函数、类、变量，也可以包含可运行的代码。也是 .py 文件

用 import 语句来导入模块。from…import 语句可以在模块里选择特定的项目，而不是加载整个模块。加 * 导入所有名字。（跟直接 import 有什么区别？）

将模块作为脚本运行

所有模块都有个内置属性 __name__，这个东西在第一篇笔记开始提到过。当模块被导入时，这个属性的值会被设为模块的名字；而当模块作为脚本直接运行的时候，这个值被设为 __main__。这样就有了一个妙用，可以让某些测试语句只在模块作为脚本时运行

添加这条判断语句后，在命令行中 import 模块不会执行测试代码，直接运行则会。

模块放哪了？

Python 解释器会按以下顺序搜索要导入的模块

  • 当前目录
  • PYTHONPATH
  • 安装默认路径

命名空间和范围

变量是映射到对象的名字，命名空间是一个存储（名字，对象）的字典。

重名时优先局部变量

使用 global 告诉 Python 这是一个全局变量，不要在局部命名空间搜索了

相关函数

  • dir(module), 返回模块中所有的模块，变量，函数的名字，用列表
  • globals() and locals(), 返回作用范围内的全局和局部变量，字典，可以用 key() 提取所有名称
  • reload(module), 模块导入到脚本里，顶层代码只执行一次，此函数可以用于重复执行

package 就是一个文件夹，里面可以包含多个模块，要有一个 __init__.py 配置文件，说明包调用了哪些模块的哪些功能，注意 Python 3 的语法。
