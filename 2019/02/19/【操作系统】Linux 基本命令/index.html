Linux 基本命令

发表于 2019-02-19 更新于 2022-08-09 分类于 操作系统 Disqus：

  1. cd ‘~’ 代表 home

  2. ls list

    1. ls -l long
    2. ls -a all
    3. ls -lh human

  3. touch 创建文件

  1. cp old new(or folder)

    cp -i interact

    cp -R folder1 folder2 recursively

    cp file* folder/ 以 file 开头的所有文件, *4 结尾

    cp file1 file2 folder1/

  2. mv file1 folder1/ 剪切 move

    mv file1 filerename 重命名

  3. mkdir folder3 创建文件夹

    mkdir folder3/f3 多文件夹

    rmdir 只能移除空文件夹

    rm file1

    rm -I 三个以上提醒是否删除

    rm -r folder3 recursively 删除文件夹和其中所有内容

  4. nano 编辑器

  5. cat 显示内容(放到屏幕上)

    -n line number

    cat t.py > t1.py (t 到 t1)

    cat t.py t1.py > t2.py 合并到 t2

    cat t3 >> t2 补充到后面

  6. od 打开非文本文件

  7. grep 查找文件里符合条件的字符串

  8. 权限

    type user group other

    chmod u+r t1.py change mode, user + read 权限

    chmod u-r t1.py

    chmod a-r t1.py

    chmod ug+r t1.py
